Panduan Outlook (Windows)

1 Generate & Salin Signature
1Buka browser Anda dan akses link generator: https://email-signature-sand.vercel.app/
2Pada panel Konfigurasi di sebelah kiri, pilih template (contoh: BINA NUSANTARA), lalu isi Informasi Pribadi Anda seperti Nama, Jabatan, dll.
3Anda dapat melihat hasil secara real-time pada panel Preview Signature.
4Jika sudah sesuai, klik tombol biru "Salin Signature". (Format teks dan HTML akan otomatis tersalin ke clipboard perangkat Anda).
2 Tambah Signature di Outlook (Windows)
1Buka aplikasi Microsoft Outlook.
2Masuk ke Settings dengan mengklik ikon ⚙️ (gear), biasanya terletak di pojok kanan atas layar.
3Navigasi ke menu Accounts atau Mail, lalu cari dan pilih opsi Signatures.
4Pada halaman Signatures, klik tombol biru + Add signature untuk membuat signature baru.
5Akan muncul kotak editor. Pertama, ketik Nama Signature di kolom judul (contoh: "Main Signature").
6Klik pada kotak teks kosong di bawahnya, lalu tekan Ctrl + V (atau klik kanan dan pilih Paste) untuk menempelkan signature dari website.
💡 Penting: Jika muncul ikon opsi tempel kecil setelah paste, pastikan pilih Keep Source Formatting agar layout dan logo tetap utuh.
7Setelah signature muncul di editor, klik tombol Save di dalam editor.
8Kembali ke halaman utama Signatures dan atur dua dropdown berikut:
•Default for new messages: Klik dropdown dan pilih nama signature baru Anda. Ini memastikan signature otomatis muncul setiap kali Anda menulis email baru.
•Default for replies and forwards: Klik dropdown dan pilih signature yang sama jika Anda ingin signature muncul saat membalas atau meneruskan email.
9Pastikan klik tombol Save di jendela settings untuk menerapkan semua perubahan.

Panduan Outlook (Mac)

1 Generate & Salin Signature
1Buka browser Anda dan akses link generator: https://email-signature-sand.vercel.app/
2Pada panel Konfigurasi di sebelah kiri, pilih template (contoh: BINA NUSANTARA), lalu isi Informasi Pribadi Anda seperti Nama, Jabatan, dll.
3Anda dapat melihat hasil secara real-time pada panel Preview Signature.
4Jika sudah sesuai, klik tombol biru "Salin Signature". (Format teks dan HTML akan otomatis tersalin ke clipboard perangkat Anda).
2 Tambah Signature di Outlook (Mac)
1Buka aplikasi Microsoft Outlook.
2Di menu bar atas layar (di samping logo Apple), klik Outlook, lalu pilih Settings... (atau Preferences pada versi lama).
3Pilih menu Signatures.
4Klik ikon + (plus) atau tombol + Add signature untuk membuat signature baru.
5Beri nama signature Anda (contoh: "Main Signature").
6Klik di dalam area teks, lalu tekan Cmd + V untuk menempelkan signature dari website. Pastikan format sumber dipertahankan (Keep Source Formatting).
7Simpan signature tersebut.
8Sama seperti di Windows, pastikan Anda mengatur bagian Default signatures:
•Pilih akun email Anda (jika diminta).
•Untuk New messages, pilih signature yang baru dibuat.
•Untuk Replies/forwards, pilih signature yang sama.
9Tutup jendela pengaturan.
